Ground Telemetry¶

The Ground Telemetry communicates with the Avionics and relays the telemetry data to the Ground Computer.

Inputs¶

  • Avionics Telemetry (RF)

Outputs¶

  • Ground Computer (USB)

Block Properties¶

Antenna: Yagi antenna Cable: Coaxial with BNC to Coaxial adapters Telemetry Interface: TeleDongle
